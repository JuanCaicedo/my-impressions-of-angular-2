### How to make a component ``` @Component({ // specify selector and dependencies }) @View({ // specify template }) class TitleComponent { // make object that controls component } ```
Event binding ``` <tag (eventName)="fnToCall()"></tag> ```
Module loader (System.js) ``` var dependencies = [ ... ] dependencies.forEach(function(dependency){ System.import(dependency); }); ``` And in each dependency ``` bootstrap(NameOfComponent); ```
No need for "$scope"! ``` <h1 >{{propertyOnComponent}}</h1> ```
Dynamic html properties ``` <img [src]="selectedMember.avatar_url"></img> ```
